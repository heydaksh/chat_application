# 💬 Flutter Chat App

## 📌 Overview

This is a chat application built using Flutter.
The main focus of this project is **UI development and API integration**.

---

## 🚀 What I Did

* Designed and developed the complete **chat UI** using Flutter
* Integrated **ready-made APIs** into the application
* Used **WebSockets** for real-time communication
* Handled API responses and connected them with the UI
* Implemented features like chat screen, user list, and messaging flow, friend add/remove. Friend suggestion.
* Local Authentication using Email/phone no.
* Added Forced LoggOut function. 

---

## ⚙️ Backend

* The backend is **NOT developed by me**
* It is already built using **Node.js**
* I only used the provided APIs and integrated them into the app
* Real-time messaging is powered using **WebSockets**

---

## 🛠️ Tech Stack

* **Frontend:** Flutter (Dart)
* **Backend:** Node.js (pre-built APIs)
* **Real-time Communication:** WebSockets
* **State Management:** GetX

---

## 📷 Features

* Real-time chat using WebSockets
* User-friendly interface
* API-based messaging system
* Clean and responsive design
